# 📂 Assets / Images folder
# 
# This folder is where ALL your project images should go.
#
# Suggested files:
#   logo.png          → Your site logo (shown in top-left of sidebar)
#   hero-bg.jpg       → Hero/banner background on the dashboard
#   avatar.jpg        → Your profile photo (shown in top navbar)
#   package-1.jpg     → Package card image for Goa
#   package-2.jpg     → Package card image for Manali
#   package-3.jpg     → Package card image for Kerala
#   ...and so on
#
# How to use an image in HTML:
#   <img src="assets/images/logo.png" alt="Logo" />
#
# After adding images, search for "📂 IMAGE SLOT" comments in the HTML files
# to find exactly where to uncomment the <img> tags.
